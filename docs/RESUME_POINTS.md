# 简历项目描述（可直接粘贴）

## 项目名称

山东麻将 AI 助手（Web 交互 + 规则引擎 + LLM 策略解释）

## 一句话版本

独立完成一个面向山东麻将教学与实战辅助的系统，前端实现互动牌局与策略可视化，后端实现规则引擎与自对弈数据生成，并接入 LLM 提供结构化决策建议。

## 三条版本（推荐放简历）

1. 设计并实现 React + TypeScript 前端，构建“摸牌-打牌-等待”三阶段互动牌局，支持财神、碰/杠、弃牌流等状态录入与实时展示。  
2. 基于 Python 实现山东麻将规则引擎与策略脚本，覆盖胡牌判定、自对弈数据生成、参数调优流程，形成可复现的实验基础。  
3. 接入 OpenAI 风格接口并约束 LLM 输出为 JSON，联动牌局上下文生成推荐舍牌、风险等级与动作建议，提升策略解释性与可交互性。  

## 面试扩展点

- 工程侧：如何保证前后端规则一致性（27 种数牌、财神、胡牌形态定义）
- 算法侧：如何在可解释规则与模型能力之间做取舍
- 产品侧：为什么采用“手工录入 + 实时建议”的教学型交互路径
